## 初识VUE

```vue
<script type="text/javascript" src="../js/vue.js"></script> 引入框架

<script type="text/javascript">
        Vue.config.productionTip=false; // 阻止vue在启动时生成生产提示

        //创建vue实例
        const x = new Vue({
            el:'#root' , //用于指定当前vue实例为哪个容器服务
            data:{  // data中用于存储供el所指定的容器使用的数据
            }，
            methods:{
            	// 各种方法
        	}
        });
 </script>
```

## Vue模板语法

```
插值语法	{{key}}
指定语法	v-bind | v-model | v-for | v-if | ...
```

## 数据绑定

```vue
单向数据绑定	v-bind:value="name"	===> :value='name'
双向数据绑定	v-model:value="name" ===> v-model='name'
```

## el与data的两种写法

```vue
el第一种：	el:'#root'
  第二种：	vm.$mount('#root')

data第一种：	data:{
				数据
			}
	第二种：	data:function(){
				return {
					数据
				}
			}
```

## MVVM模型

```vue
		M: (Model) : 对应data中的数据
        V: (View)  : 模板
        VM：(ViewModel) ： Vue实例对象
```
