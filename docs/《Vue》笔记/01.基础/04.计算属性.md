## 计算属性

```vue
computed:{
       //完整写法
      fullname : {
        // 当有人读取fullname时，get被调用，返回值作为fullname的值
        //调用时机：初次读取fullName时、所依赖的数据发生变化时 
        get(){
            return this.x + '-' +this.m
        },
        // // 当fullname被修改时调用    问题：不设置这个也可以修改f，并且不通过set函数
        set(value){
            const arr = value.split('-')
            this.x=arr[0]
            this.m=arr[1]
        }
    } 

	// 简写形式	只有读取，没有赋值功能
	fullname(){
		return this.x + '-' +this.m
	}
}

```
