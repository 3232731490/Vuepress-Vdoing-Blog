## 过滤器

### 1、语法

```html
{{数据 | 过滤器1 | 过滤器2 | ...}}
```

### 2、案例

```html
 		<!--过滤器实现-->
        <h3>现在是： {{date | timefmt}}</h3>

        <!--过滤器传参  过滤器总是会将 | 前面的参数传进过滤器函数-->
        <h3>现在是： {{date | timefmt('YYYY_MM_DD')}}</h3>

        <!--多个过滤器串联-->
        <h3>现在是： {{date | timefmt('YYYY_MM_DD') | myslice}}</h3>
```

```javascript
			// 全局过滤器
            Vue.filter('myslice',function(value){
                return value.slice(0,4);
            })

			// 局部过滤器
            filters:{
                timefmt(value , str='YYYY-MM-DD HH:mm:ss'){
                    return dayjs(value).format(str)
                },
                myslice(value){
                    return value.slice(0,4);
                } 
            }
```
