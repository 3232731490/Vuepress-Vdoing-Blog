## 收集表单数据

```html
    <div id="root">
        <!--
            v-model 收集input的value值，在单选框多选框中要手动指定value,收集多选框时还需要手动设置接受数据类型为数组
        -->
        <form @submit.prevent='sub'> 
            <label for="admin">账号： </label>
            <!-- v-model.trim  去除前后空格-->
            <input type="text" id='admin' v-model.trim='userInfo.admin'><br/><br/>
            <label for="pwd">密码： </label>
            <input type="password" id='pwd' v-model='userInfo.pwd'><br/><br/>
            <label for="age">年龄： </label>
            <!-- type="number"限制输入数字  v-model.number： 将输入转成数字-->
            <input type="number" id='age' v-model.number='userInfo.age'><br/><br/>
            性别: 
            男<input type="radio" name='sex' v-model='userInfo.sex' value='male'>
            女<input type="radio" name='sex' v-model='userInfo.sex' value='female'><br/><br/>
            爱好：
            Study<input type="checkbox" v-model='userInfo.hobby' value='Study'>
            Game<input type="checkbox" v-model='userInfo.hobby' value='Game'>
            Sleep<input type="checkbox" v-model='userInfo.hobby' value='Sleep'><br/><br/>
            所属校区：
            <select v-model='userInfo.city'>
                <option value="">请选择校区</option>
                <option value="FengXian">奉贤</option>
                <option value="XuHui">徐汇</option>
            </select><br/><br/>
            其他信息
            <!-- v-model.lazy : 失去焦点时再收集信息-->
            <textarea v-model.lazy='userInfo.other'></textarea><br/><br/>
            <input type="checkbox" v-model='userInfo.agree'>阅读并接受协议 <a href="http://www.baidu.com">用户协议</a>
            <input type="submit" value="提交">
        </form>
    </div>
```

```javascript
	<script type='text/javascript'>
        new Vue({
            el:'#root ',
            data:{
                userInfo:{
                    pwd:"",
                    admin:"",
                    sex:"",
                    age:0,
                    hobby:[],
                    city:'FengXian',
                    other:"",
                    agree:false
                }
            },
            methods: {
                sub(){
                    console.log(this._data)
                }
            },
        })
    </script>
```
