## 条件渲染

```html
 		<!--使用v-show做条件渲染  如果表达式为假，则页面中隐藏元素-->
        <h2 v-show="true">欢迎您{{name}}</h2>
        <h2 v-show="1===1">欢迎您{{name}}</h2>

        <!--使用v-if做条件渲染  如果表达式为假，则页面中直接删除元素-->
        <!--
            v-if
            v-else-if
            v-else
        -->

		<!--template只能和v-if配合使用，不会影响结构，在页面渲染时不会把template渲染进页面-->
        <template v-if='1===1'>
            <h2>您好</h2>
            <h2>{{name}}</h2>
        </template>
```
