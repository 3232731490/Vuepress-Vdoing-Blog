## 事件处理

### 1. 语法形式

```html
		<!--绑定事件-->
        <button v-on:click = "showInfo">点我提示信息</button>
        <!--简写1  不传参数-->
        <button @click="showInfo1">点我提示信息1</button>
        <!--简写2  传参数-->
        <button @click="showInfo2(66,$event)">点我提示信息2</button>
```

### 2. Vue的事件修饰符

```html
        <!--
            Vue的事件修饰符
                prevent ： 阻止默认事件
                stop    :  阻止事件冒泡
                once    ： 事件只触发一次
                capture :  使用事件的捕获模式
                self    :  只有event.target是当前操作元素时才触发事件
                passive ： 时间的默认行为立即执行，无需等待事件回调执行完毕
            修饰符可连写
                @click.prevent.stop
        -->
```

### 3. 常用按键事件别名

```html
        <!--
            Vue 常用按键别名
                enter，delete，esc，space，tab(特殊，必须配合keydown使用)，up，down，left,right
            按键也可以连写
                @keyup.ctrl.y
        -->
案例		<input type="text" placeholder="按下回车提示输入" @keyup.enter = "showInfo4">
```
