## 浏览器本地缓存webStorage

```java
      1、存储内容大小一般为5MB左右
      2、浏览器铜棍Window.sessionStorage,Window.localStorage属性实现本地存储机制
      3、相关API(sessionStorage与localStorage方法名都一致,如下省略)：
            1）setItem('key','value')
            2）getItem('key')
            3)removeItem('key')
            4)clear()
      4、备注：
            1）SessionStorage存储的内容会随着浏览窗口器关闭而消失
            2）LocalStorage存储的内容需要手动清除才可消失
            3）如果某个key对应的value获取不到，则getItem返回null
            4）JSON.parse(null) === null
```
