## 脚手架文件结构

```java
|-- node_modules 各种模块库
|-- .gitignore', git版本管制忽略的配置
|-- babel.config.js',    babel的配置文件
|-- package-lock.json',  包版本控制文件
|-- package.json',   应用包配置文件
|-- vue.config.js',  修改默认配置
|-- public',
|   |-- favicon.ico',    页签图标
|   |-- index.html',     主页面
|-- src',    
    |-- App.vue',    汇总所有组件
    |-- mymain.js',  入口文件
    |-- assets',     存放静态资源
    |   |-- logo.png',   图标
    |-- components', 所有组件
        |-- School.vue',
        |-- Student.vue',
```

##  不同vue版本

```java
      runtime.xxx.js 只包含核心功能，没有模板解析器
```

      新建一个vue.config.js可以修改默认配置
